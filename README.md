# 💰 GameOver Shop - Калькулятор ЗП

Интерактивное приложение для расчёта заработной платы сотрудников с прогрессивной шкалой мотивации.

## 🎯 Цели проекта

- **Прозрачность** — сотрудник видит как считается его ЗП
- **Мотивация** — визуализация "сколько ещё допродать чтобы перейти на новый уровень"
- **Простота** — понятный интерфейс без сложных формул
- **Масштабируемость** — легко добавить новые роли (директор, продавцы ЦУМ и т.д.)

## 🛠 Технологический стек

- **Next.js 14** (App Router) — React фреймворк
- **TypeScript** — типизация
- **Tailwind CSS** — стилизация
- **Shadcn/ui** — UI компоненты
- **Recharts** — графики прогресса
- **Framer Motion** — анимации

## 💡 Фичи

1. **Калькулятор ЗП** — ввод суммы продаж → мгновенный расчёт
2. **Визуализация уровней** — прогресс-бар до следующего уровня
3. **Мотивационные подсказки** — "Ещё 300к и ты на уровне Мастер!"
4. **Детальная разбивка** — откуда берётся каждая часть ЗП
5. **Сравнение сценариев** — "А что если продам на 500к больше?"
6. **История расчётов** — сохранение предыдущих расчётов
